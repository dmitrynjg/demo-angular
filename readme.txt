Для запуска frontend необходимо войти в папку кодом через терминал, установить все пакеты через команду npm i и написать команду npm run start. В терминале будет указан порт, на котором запущен сервер (обычно это 4200 порт, если он не занят). После этого вы сможете открыть frontend в браузере по адресу http://127.0.0.1:4200/. Если вы меняете порт в backend, то вам нужно будет зайти в файл proxy.conf.json и изменить "target": "http://127.0.0.1:3000/api" на "target": "http://127.0.0.1:port/api", где port - заданный вами порт на backend.
